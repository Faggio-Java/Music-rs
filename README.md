Simple music player in your terminal
<br>
Controls <br>
P = Pause <br>
O = Resume <br>
Up/Down Arrow Keys = Navigation <br>
Enter = Play Selected song <br>
This scans the $HOME/Music folder so make sure to play everything in there. <br>
